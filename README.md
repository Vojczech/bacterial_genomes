## Short read assembly
This bash script takes two FASTQ files from short read sequencing of bacterial genome. It uses Unicycler to assemble reads, filters out phiX contamination and uses Prokka to annotate genes.

## Software used
[Unicycler assembler](https://github.com/rrwick/Unicycler)
[SPAdes assembler](http://cab.spbu.ru/software/spades/)
[Prokka](https://github.com/tseemann/prokka)
[bioawk](https://github.com/lh3/bioawk)
[pullseq](https://github.com/bcthomas/pullseq)
